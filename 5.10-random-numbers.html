5.10 Random Numbers

Notes

Random number are useful in software devlopment for many reasons-game mechanics generating usernames/passwrods, anytime you need something seemingly unpredictable

Pseudo-random numbers- computers are just big calculators solving formulas and can't create truly random number, but instead formulas use an algorithm the generates numbers that seem random

Randon Numbers in JS

  • math.random() is a function that returns a psuedo-random number between 0 and 1, not including 1
  • math.floor is a funtion taht takes a decimal number and rounds it down-chops off the decimal part

    • Random Number Calculations

    To produce a random integer from a min value to a max value (including the min and max) use this calculation

    Math.floor(Math.random() * (max + 1 )) + min

    Example

    Generate a random number from 2 to 10 and store it in a variable

    var ramdNum = Math.floor(Math.random() * 11) + 1;
